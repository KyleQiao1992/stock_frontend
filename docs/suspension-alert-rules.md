# 停牌预警 · 规则口径定稿（draft 2026-06-25）

> 目标：对自选股做「停牌风险」预警。核心风险通道是 **异常波动 → 严重异常波动 → 强制停牌核查**。
> 本文锁定计算口径与阈值；机器可读配置见 [`server/suspensionRules.json`](../server/suspensionRules.json)。
> ⚠️ 标记的单元格是**尚未对到交易所现行原文**的，开工前必须确认（见文末「待确认清单」）。

---

## 1. 风险分级（自选股徽章 → 这套映射）

| 等级 | 含义 | 触发条件（任一） |
|---|---|---|
| 🟢 正常 | 无异动迹象 | 各项偏离值离阈值都还远 |
| 🟡 接近普通异波 | 快触发异常波动公告 | 近3日累计偏离值达到板块阈值的 ~80% |
| 🟠 已触发普通异波 / 接近严重异波 | 需披露异动公告；再走高就核查 | 已触发普通异波，或 10日/30日累计偏离值达严重档的 ~80% |
| 🔴 高停牌核查风险 | 已达严重异常波动标准 | 命中任一严重异常波动情形 → 交易所**可要求停牌核查** |
| ⛔ 已公告停牌 | 公司已公告将停牌 | 命中东财停复牌 feed |

> 文案统一用「**高停牌核查风险**」而非「必停牌」：触发严重异常波动后，规则是公司应核查披露、交易所**可**要求停牌核查，并非 100% 机械停牌。

## 3. 偏离值计算口径（沪深一致）

**收盘价涨跌幅偏离值（累计，整段窗口一次算）：**

```
偏离值 = (个股期末收盘价 / 个股期初前收盘价 - 1) × 100%
       - (对应指数期末收盘点数 / 对应指数期初前收盘点数 - 1) × 100%
```

实现要点：
- **用不复权日 K**（官方涨跌幅口径），不能用后复权序列——除权日会算错。
- 窗口按**交易日**计，不是自然日。
- **停牌日不计入窗口**，跨过停牌日继续向前取交易日。
- 「最近一轮」普通异波是**有状态**的：发布异动公告后从触发日重新起算 → 需持久化每只票「上一次触发日」，每日跑批回溯。

**对应基准指数：**

| 板块 | 指数 | 代码 | 置信度 |
|---|---|---|---|
| 沪市主板 | 上证指数 | `000001` | 多源一致 ✅ |
| 创业板 | 创业板综指 | `399102` | 多源一致 ✅ |
| 深市主板 | 深证A股指数 | `399107` | 深交所《交易规则》对应分类指数 ✅ |
| 科创板 | 科创50（上证科创板50成份指数） | `000688` | 上交所公告口径 ✅ |

## 4. 异常波动（普通）认定标准

连续 **3 个交易日**内收盘价涨跌幅偏离值累计达到下表阈值，即触发，需披露异动公告。

| 板块 | 普通股阈值 | ST 股阈值 | 量/价其他情形 |
|---|---|---|---|
| 沪市主板 | ±20% | ±12%（现行）→ ±20%（2026-07-06起） | 换手率：日均换手率/前5日 ≥30倍 且 累计换手率 ≥20% |
| 深市主板 | ±20% | ±12%（现行）→ ±20%（2026-07-06起） | 同上 + 振幅类 |
| 创业板 | ±30% | — | 单日偏离 / 换手率 / 振幅 |
| 科创板 | ±30% | — | 同上 |

> ST 阈值：主板 ST/\*ST 涨跌幅限制由 5%→10% 自 **2026-07-06** 实施，异动偏离值阈值同步由 ±12% 调到 ±20%。当前（2026-06）仍为 ±12%，引擎按日期切换（见 `suspensionRules.json` 的 `stThresholdChange`）。

## 5. 严重异常波动认定标准 → 强制核查/停牌核查

命中任一情形即属严重异常波动；公司应次日核查披露，交易所**可要求停牌核查**。

| 板块 | 10日累计偏离 | 30日累计偏离 | 次数类情形 |
|---|---|---|---|
| 沪/深主板 | **+100% / −50%** | **+200% / −70%** | 10个交易日内 **4次同向异常波动**（每次 = 一轮3日±20%；媒体所称"10日4次同向涨停"即此条简化） |
| 创业板/科创板 | **+100% / −50%** | **+200% / −70%** | 10个交易日内 **3次同向异常波动**（每次 = 一轮3日±30%） |

注意：**涨跌阈值不对称**——涨向 +100%/+200%，跌向 −50%/−70%。
指标**自交易所公布的次一交易日或复牌之日起重新计算**（对应"轮"的重置，需持久化每只票上一次触发日）。

## 6. 其他停牌风险（叠加，非偏离值类）

- **盘中临时停牌**：仅无涨跌幅限制票/新股，较开盘价首次 ±30% 停10分钟、±60% 停到14:57。需实时分时数据，本期可不做或后置。
- **计划性停牌**（重组/重大事项）：不可预测，靠东财停复牌 feed（`stock_tfp_em` → `datacenter-web.eastmoney.com/api/data/v1/get`，`RPT_CUSTOM_SUSPEND_DATA_INTERFACE`）公告后提前提醒。feed 是**前瞻**的，能拿到「N日后起停牌」。
